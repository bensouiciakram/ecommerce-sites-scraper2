## Introduction

This project is based on the Scrapy framework and is designed to scrape and extract product-related information from e-commerce websites. It extracts fields such as `product_url`, `image_url`, `title`, `brand`, `high_price`, `low_price`, and `sku` and saves the information into an `output.csv` file.

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Python 3.x
- Scrapy
- playwright
- nested_lookup
